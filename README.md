# Book Tracker

## Contributor

- @samseburn (고석영)

## Tech Requirement (Tech Stack)

- Create-react-app
- React.js
- react-router-dom
- react-hook-form
- Context API
- TypeScript
- ESLint, Prettier

## Script

```
$ npm install
$ npm run start
```

```
$ npm run build
$ npm run deploy
```

### 주요 기능

1. 독서 기록 CRUD
2. Context API를 이용하여 상태 관리
3. Firebase를 연결하여 데이터 저장
4. react-hook-form으로 마이그레이션 진행(포스트)
5. recoil로 상태관리(포스트)
6. Atomic 디자인으로 컴포넌트 구조 설계(포스트)
